<h1>LeffaAppi</h1>

<p>Frontend available at: <a href="https://leffa-frontend.onrender.com">https://leffa-frontend.onrender.com</a></p>

<p>This project is a fullstack web application developed in <strong>Autumn 2025</strong> as part of the <em>Web Programming Application Project</em> course at <strong>Oulu University of Applied Sciences</strong>.</p> 

<p>This project was developed by students of Information Technology:</p>

  <li><a href="https://github.com/JoelLamsi">JoelLamsi</a></li>
  <li><a href="https://github.com/Mahiro-dev">Mahiro-dev</a></li>
  <li><a href="https://github.com/tup4kka1">tup4kka1</a></li>
</ul>

<p>The application utilizes the <strong>TMDB (The Movie Database) open API</strong> to fetch movie data. Browsing movies does not require authentication, but registered users gain access to social features such as groups and favorites.</p>

<p>After deleting account it can be restored by signing in again within <strong>14 days</strong>. After 14 days all associated data is permanently deleted, including: user profile data, reviews, favorites, group memeberships, and profile image stored in the external R2 service.</p>
